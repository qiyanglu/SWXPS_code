# PLANS.md

A plan is required for any multi-step implementation.

Each plan must include:

## Goal

What physical or coding capability will be added?

## Physics background

Define all relevant physical quantities and equations.

## Files to create or modify

List expected files.

## Implementation steps

Break the task into small milestones.

## Tests

List tests that must pass.

## Validation

Describe the expected physical behavior.

## Progress log

Record what has been done and what remains.
